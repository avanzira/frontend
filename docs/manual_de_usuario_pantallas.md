# Manual de Usuario — Pantalla a Pantalla

Este manual explica cada pantalla de forma sencilla, paso a paso.

## 1) Inicio de sesion
Entra con tu usuario y contrasena.
- Escribe tu usuario.
- Escribe tu contrasena.
- Pulsa "Iniciar sesion".

Si la contrasena es incorrecta, veras un aviso.

## 2) Barra superior (menu)
En todas las pantallas veras una barra arriba con accesos:
- Inicio: vuelve al panel principal.
- Entidades: abre la gestion de datos basicos.
- Documentos: abre los documentos del negocio.
- Informes: abre las tablas de consulta.
- Perfil: abre tus datos personales.
- Salir: cierra la sesion.

Si no eres ADMIN, no veras la seccion de usuarios.

## 3) Panel principal (Inicio)
Aquí eliges una de las 3 secciones principales:
- Entidades
- Documentos
- Informes

Pulsa sobre una opcion para entrar.

## 4) Perfil
En esta pantalla puedes:
- Cambiar tu usuario o email.
- Elegir idioma y tema de color.
- Cambiar tu contrasena.

Cuando guardas, los cambios se aplican al instante.
La contrasena debe tener al menos 6 caracteres.

## 5) Entidades
Aquí estan los datos basicos:
- Usuarios (solo ADMIN)
- Productos
- Clientes
- Proveedores

Pulsa la opcion que quieras gestionar.

## 6) Usuarios (solo ADMIN)
Esta pantalla es solo para administradores.

### Ver lista
Veras una tabla con los usuarios existentes.
Puedes filtrar por:
- Rol (ADMIN/USER)
- Estado (activos/inactivos)

### Crear usuario
Rellena el formulario con:
- Usuario
- Email (opcional)
- Rol
- Idioma (opcional)
- Tema (opcional)
- Contrasena

Pulsa "Guardar".
El nuevo usuario aparecera en la tabla.

### Editar usuario
En la tabla, pulsa "Editar".
Modifica los datos y pulsa "Guardar".

### Desactivar usuario
En la tabla, pulsa "Desactivar".
El usuario no se elimina, queda inactivo.

### Restaurar usuario
Si un usuario esta inactivo, pulsa "Restaurar".
Volvera a estar activo.

### Cambiar contrasena
Usa el formulario de cambio de contrasena.
Esto sirve para tu propia cuenta.

## 7) Productos

### Ver lista
Veras productos con:
- Nombre
- Unidad
- Inventariable (si/no)
- Costo promedio (solo lectura)

Puedes filtrar por:
- Nombre
- Inventariable (si/no)

### Crear producto
Rellena:
- Nombre
- Unidad
- Inventariable
Pulsa "Guardar".

### Editar producto
Pulsa "Editar", cambia datos y guarda.

### Desactivar producto
Pulsa "Desactivar" para dejarlo inactivo.

### Restaurar producto
Pulsa "Restaurar" para volver a activarlo.
